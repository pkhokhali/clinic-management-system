# Assets Folder

This folder should contain:

1. **icon.png** - App icon (1024x1024px)
2. **splash.png** - Splash screen image (1242x2436px recommended)
3. **adaptive-icon.png** - Android adaptive icon (optional, uses icon.png if not provided)
4. **favicon.png** - Web favicon (optional, uses icon.png if not provided)

## Temporary Solution

For now, you can use any placeholder image or create simple ones:

- Create a 1024x1024px PNG file named `icon.png`
- Create a 1242x2436px PNG file named `splash.png`

Or use online tools to generate them:
- https://www.appicon.co/
- https://appicon.build/

## For Quick Testing

You can temporarily use simple colored squares or download placeholder images from:
- https://placeholder.com/

The app will work without these assets, but they're required for building production versions.
